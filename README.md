### Running project locally
#### 1. install project's dependencies
```sh
yarn install
```
or
```sh
npm install
```

#### 2. run project
```sh
yarn start
```
or
```sh
npm run start
```

#### 3. access at
[localhost:7777](http://localhost:7777/)
